## Installation

```bash
omarchy-theme-install https://github.com/avis3nna/desert-gleam
```

## Extended App Support

For theming in Vesktop (Discord client), Spotify, and other apps, install [omarchy-theme-hook](https://github.com/imbypass/omarchy-theme-hook).
